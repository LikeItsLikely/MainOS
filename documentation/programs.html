How to add a program and what properties to use

Under C:/mainos you can find a file called customprograms.txt. This is basically your custom repository. You can add a program here. Your custom repository has to be a valid JSON, else your entire custom repository won't be loaded.

A program can have the following properties:
id - The ID of your program (string) title - The window title of your program (string) icon - The path of the icon for your program (string) src - The path, your program is stored at (string) sandbox - If your program should run in a sandbox. Possible values are 0,1,2 where 0 is not sandboxed at all and 2 is very sandboxed (0,1,2) spawnicon - whether an icon should appear on the desktop or not (0,1) autostart - whether a program should start automatically (0,1) noborder - wheter a program should have a border (0,1) Example: { "minecraft": { "id": "minecraft", "title": "Minecraft Classic", "icon": "https://www.minecraft.net/etc.clientlibs/minecraft/clientlibs/main/resources/android-icon-192x192.png", "src": "https://classic.minecraft.net/", "sandbox":1 } }
